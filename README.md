# Bootstrap



Ubuntu:
```bash
& PYTHON_ENV_FOR_APP=venv
$ virtualenv $(PYTHON_ENV_FOR_APP)
$ . $(PYTHON_ENV_FOR_APP)/bin/activate
$ pip install -r requirements.txt
$ python bootstrap.py
```

# Running

```bash
$ python app.py
```
